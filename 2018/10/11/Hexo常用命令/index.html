HEXO常用命令

发表于 2018-10-11 | 分类于 笔记 |
字数统计: 278 | 阅读时长 ≈ 1

久一点不写博客一些命令又模糊了，还是记下来吧，找的方便，嘿嘿嘿(‾◡◝)。

命令总结

init

$ hexo init [folder]

初始化，新建网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

new

$ hexo new [layout] <title>

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

generate

$ hexo generate

生成静态文件。可简写为：

$ hexo g

publish

$ hexo publish [layout] <filename>

发表草稿。（没怎么用过）

server

$ hexo server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。

deploy

$ hexo deploy

部署网站。可简写为：

$ hexo d

clean

$ hexo clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)。
在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令

list

$ hexo list <type>

列出网站资料。

version

$ hexo version

显示Hexo版本。

显示草稿

$ hexo --draft

显示source/_drafts文件夹中的草稿文章。
