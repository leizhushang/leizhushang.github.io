插画笔记

发表于 2019-12-05 | 分类于 学习 |
字数统计: 2,851 | 阅读时长 ≈ 10

看视频总是分心走神，做下笔记记录下~~

第一课 型体

要点：

  1. 通过点线（点对点）、分析临摹人物插画，锻炼形体把控能力。
  2. 脑洞大开：描摹概括形体，用脑洞或实物替换人物造型，锻炼透视形体能力。

课上示范：

  1. 临摹插画人物。
  2. 描摹人物形体，换装（内衣）。

第二课 人物五官转向及透视

作业讲解：

  1. 线条问题、毛边、断续、太细、太软（无力度，无型体转折）、粗细不一
  2. 型不准

要点：

  1. “三庭五眼”最舒适耐看,三庭：发际线、眉心、鼻尖、下巴；五眼：五个眼睛的宽度
  2. 像 != 好
  3. 十字交叉线、横线穿过眼睛耳朵、竖线穿过鼻子嘴巴
  4. 竖线控制左右（左转右转），横线控制上下（抬头低头）
  5. 近大远小（大小和间距）
  6. 轴线随视线（左上、左下、右上、右下）发生移动变化

课上示范：

  1. 找中轴线，加十字交叉线。
  2. 画中轴线，加五官。

第三课 不同风格的五官表现及头发绘制（十字交叉线的至高点/头部转向）

作业讲解：

  1. 线条问题不流畅、毛边。
  2. 十字交叉线横线太直，没有弧度就没有透视。
  3. 透视不对（眼睛大小、眼间距、耳朵位置）
  4. 耳朵要跟着头/五官转
  5. 五官不协调（大小、位置）
  6. 中轴线/十字线分析不对

要点：

  1. 至高点、至低点一定是十字交叉线的交点（方便寻找确定眼睛高度位置、鼻子、确定透视）
  2. 五官调整影响性格的变化
  3. 头发绘制技巧：分块、区分走向、细致化
  4. 鬓角（耳朵边）处有头发
  5. 细化头发绘制时注意头发成组绘制，越顺直的区域线条越少

课上示范：

  1. “布玛”五官调整对性格感官的影响
  2. 头发绘制示意：1）确定大体发型，2)分块，3）确定走向，4）细化

第四课 面部表情及转向/照片转头发

作业讲解：

  1. 线条问题
  2. 五官透视（近大远小：眼睛大小、眼间距长短、耳朵透视）
  3. 五官比例问题
  4. 头型图歪歪扭扭

要点：

  1. 脖子的起始点在十字交叉线与头部相交点附近 。
  2. 同一人物的转向。
  3. 头发转折处、起始处、结束处加重画笔力度（有轻有重）。

课上示范：

  1. 脸部五官转向
  2. 图片、照片头发分析绘制
  3. 改变表情：改变眉毛和嘴巴

第五课 人物身体结构及不同年龄的人物比例

作业讲解：

  1. 线条不流畅
  2. 头发的转向和透视
  3. 脸型不饱满
  4. 人物五官组合不好看

要点：

  1. 四肢三骨（三骨：头、胸腔骨、盆骨）
  2. 头的大小影响头身比进而影响感官高度
  3. 插画中儿童头身比为2~3头身，青少年时期为3~5头身，成年人时期为4~6头身（至多6.5头身）
  4. 换脸法：1）提取框架，2）换脸、换衣服
  5. 2~4头身的人物不能有明显的肩膀
  6. 3头身以上人物可以有脖子

课上示范：

  1. 换脸法示意

第六课 人物动态

作业讲解：

  1. 线条问题
  2. 人物比例不准
  3. 人物五官透视（头部、眼睛、耳朵、头发）
  4. 人物细节（服装）
  5. 五官年龄与身体比例的适配
  6. 脖子不能太粗
  7. 脸型不准不圆润
  8. 尽量多做五官组合不要照搬
  9. 腿的转向/分析人物身体

要点：

  1. 理解人物身体转向（转向透视）
  2. 动手动脚（穿插、大臂小臂长度等长）
  3. 动态线
  4. 近大远小、近长远短
  5. 领口代表身体中线、随左右转向而变化，肩膀同理
  6. 让人物侧一些，不要左右对称（解决动态死板的方法）

课上示范：

  1. 火柴人动态
  2. 人物动态分析
  3. 动态人物替换练习：1）提取人物动态，2）确定新人物的头身比

第七课 几何造型法

作业讲解：

  1. 线条问题
  2. 两手不一致
  3. 动态小人分析不对
  4. 没有动态参考
  5. 五官透视/五官间距
  6. 头发不会画
  7. 五官照搬
  8. 身体转向透视

要点：

  1. 细致，多练

课上示范：

  1. 不同几何头型的画法

第八课 原创人物的绘制

作业讲解：

  1. 线条太轻太细
  2. 五官照搬，服装未做改变
  3. 几何造型法表达太普通
  4. 五官间距及大小的把控

要点：

  1. 动态舒展
  2. 简化改变
  3. 以照片为原创素材（灵感来源）
  4. 可以保留项：1）服装穿搭，2）配饰及部分服装细节，3）性别，4）大致的动态，5）大致的发型
  5. 要做改变的：1）人物五官/头型/五官比例（参考第三课内容：打散重组），2）人物身体比例（参考第五课内容：人物比例框架），3）调整/舒展部分动态
  6. 人物原创：1）角色设定，2）风格设定

课上示范：

  1. 小丑人设原创人物
  2. 人物框架提取示范
  3. 照片人物组合插画人物

第九课 人物细节刻画

作业讲解：

  1. 线条依旧不流畅
  2. 人物五官与人物年龄不适配
  3. 五官透视
  4. 没有多个五官重组参考
  5. 五官间距
  6. 头身比例、6~6.5头身内

要点：

  1. 临摹形，点对点的观察找形，由外向内
  2. 吸色做色板:1）尽量吸干净的颜色，2）吸中间色（大面积色），3）吸亮暗变化
  3. 铺色流量开到100%
  4. 衣服绘制：控制暗面与亮面

课上示范：

  1. 临摹上色

第十课 人物细节刻画进阶

作业讲解

  1. 线条细
  2. 型不准
  3. 亮暗不够
  4. 肤色勾线用深肤色

要点：

  1. 深浅变化
  2. 各种笔刷不同效果多尝试
  3. 高光
  4. 头发：顺着头发的走向，顺毛捋

课上示范：

  1. 各种笔刷效果示范

第十一课 植物的细节刻画

作业讲解：

  1. 铺色留白
  2. 头发虚实关系
  3. 五官丑（五官间距)
  4. 十字线透视
  5. 用笔方法
  6. 颜色太亮/太艳/饱和度太高
  7. 直接抄五官
  8. 头发没有亮暗区分
  9. 勾边线颜色太淡

要点：

  1. 熟悉画笔
  2. 先吸色

课上示范：

  1. 树木
  2. 灌木树叶
  3. 露珠树

第十二课 静物的刻画

作业讲解：

  1. 色彩太丰富
  2. 模糊
  3. 太软（笔刷选择）
  4. 选色问题
  5. 细节
  6. 线条太细了

要点：

  1. 插画中的静物一定要有丰富的装饰细节：1）室内：标配花纹地毯/毛毯、强制、挂画、各种小物件；2）人物：人物的衣服一定要有装饰细节、小物件也要带装饰花纹。
  2. 三大面：1）亮面（受光面）；2）灰面（测光面）；3）暗面（背光面）
  3. 五大调：1）明暗交界线；2）反光/暗面的一部分；3）高光/物体最亮处；4）阴影；5）中间调子
  4. 扁平式的刻画方法是表现上下叠加的空间关系
  5. 刻画步骤：1）铺大色（速写铅笔：流量100%）；2）打基底/底纹（磨砂刻画）；3）绘制肌理（常用肌理、干刮肌理）；4）排线：深线/浅线（速写铅笔、高光笔）；5）冷暖灰（常用肌理笔/磨砂刻画）

课上示范：

  1. 受光球体
  2. 人物明暗高光添加
  3. 翻盖背包
  4. 椅子

第十三课 各种肌理的刻画

作业讲解：

  1. 场景中物体的大小比例
  2. 缺少装饰（小物件、小植物板件、花纹（单一））
  3. 透视/如果要表现透视、要遵循近大远小的规则
  4. 不要硬刚、要学会借助物体遮挡
  5. 室内场景一定要有地毯
  6. 不要画的太方（破形）

要点：

  1. 注意个物体对象的特质
  2. 高光
  3. 反光
  4. 透光
  5. 肌理

课上示范：

  1. 液体水珠
  2. 薄纱
  3. 木质
  4. 云
  5. 玻璃
  6. 草地
  7. 金属

第十四课 动物的绘制

作业讲解：

  1. 少考虑那么多问题，先下笔

要点：

  1. 毛发的刻画（分簇、明暗毛发）
  2. 动物：1）拟人法（换头）；2）几何图形法/象形法：a）总结动物特点；b）概括的表现

课上示范：

  1. 小兔子
  2. 奶牛
  3. 猪
  4. 恐龙
  5. 狗
  6. 羊

第十五课 构图知识

作业讲解：

  1. 出现了“头”就要注意透视

要点：

  1. 横构图：以场景为主/故事为主(远景、中景、近景)；中景近景更利于表现人物/故事
  2. 竖构图：以人物/故事为主；场景生动会说话：1）以一句话的故事出发；2）以一个/几个丰富的人设出发（由点到面）；3）凹造型：室外/室内，室内场景地上一定要有地毯，墙上一定要有墙纸，场景要铺满
  3. 对角线构图：1）画面两个主人物；2）两个主人物只需半身出现在画面对角线的两端；3）画面均衡有对话感，突出主体
  4. 圆圈式构图：1）画面中必出现多个人物（主体）；2）视觉集中，头图饱满；3）可营造较强的画面氛围（团圆）
  5. 发散式构图：1）画面从外向内延伸，利用肢体语言，有指向性；2）开放思维，突破画幅限制
  6. 舞台式构图：1）有主角光环（衬托）；2）绘图饱满，有氛围感
  7. 框架式构图：1）以第一人称视角、有镜头感；2）可以表现多层次；3）有纵深，带有一些透视；4）符合人们的视觉习惯
  8. 中心点式构图：1）大特写、只表现大脸盘子（刻画深入、眼神有话）；2）有较强的的视觉冲击力
  9. 八卦阵式构图：1）围绕中心并且对称且有均衡感
  10. 分段式构图：1）有层次感；2）可以表现情绪的起伏；3）有故事性；4）合理布局
  11. 倾斜式构图：1）让画面有流动感；2）可以借助这个构图变现情绪的起伏（不死板）
  12. 三点/三角形式构图：1）画面具有稳定性；2）画面中有3个主体；3）主体之间要有联系（做同一件事，不要凹造型）有场景氛围感
  13. 靶心式构图：1）包围感；2）突出主体衬托
  14. U型构图：1）科学留白，有形式感
  15. 视平线透视：站的越高视平线越高，看的东西越多

课上示范：

无
