MarkDown语法整理记录

发表于 2019-12-03 | 分类于 学习 |
字数统计: 312 | 阅读时长 ≈ 1

时间一久就会忘记Markdown语法，看来真的老了，记忆力真要命~~干脆直接记下来好了。
(o´・ェ・｀o)

简介

Markdown是轻量级的标记语言，使用“#”、“+”、“*”符号来进行标记，符号后至少跟一个空格才有效。

标题

在标题开头加“#”，标记标题，增加“#”个数对应标题级别增加，至多六级标题。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

列表

使用“-”、“+”、“*”标记无序列表

- Red
- Green
- Blue

引用

使用“>”标记引用字段、“<>”标记代码块等

> 一级引用
> > 二级引用
> > > 三级引用

强调

使用一个“*”或“-”标记倾斜，两个标记加粗，“~~”标记删除。

**加粗文本**
*斜体文本*
~~删除文本~~

图片与链接

图片格式： ![图片文本]（图片地址）  
链接格式： ![链接文本]（链接地址）

代码

使用“    (四个空格)”或一个制表符标记代码块，用“

分割线

使用三个以上的“***”、“---”、“___”可以建立分割线

空格

使用两个空格和回车表示换行

脚注

使用[^]标记脚注
